# Feature Selection UI

A clean, interactive UI component for users to select and customize premium software feature modules.

---

##  Description

Feature Selection UI is a modern, card-based interface designed for subscription management and feature toggling.  
It focuses on high-fidelity visual feedback, smooth interactions, and a minimalist user experience.

## Features

- **Interactive Toggles**  
  Active cards feature a distinct blue border and checked state for clear visual feedback.

- **Clean Typography**

- **Responsive Layout**  
  Fully centered, narrow-column design compatible with all screen sizes.

- **Premium Modules**
